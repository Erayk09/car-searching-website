🚗 ArabaAra Projesi

Araç Satış Platform - Frontend + Backend Uygulaması

Hazırlanan Tarih: Mart 22, 2026

1. Proje Özeti

ArabaAra, kullanıcıların araç satış ilanlarını ekleyebilmesi, görüntüleyebilmesi ve yönetebilmesi için tasarlanmış bir web platformudur.

✨ Temel Özellikler:

  • ✅ Kullanıcı Kayıt ve Giriş Sistemi (JWT Authentication)
  • ✅ Araç İlanları CRUD Operasyonları
  • ✅ Kayıt Defteri Rolü Yönetimi
  • ✅ Responsif Modern UI Tasarımı
  • ✅ RESTful API Architecture
  • ✅ SQL Server Veritabanı

2. Teknoloji Yığını

Frontend

Teknoloji Versiyon Açıklama

4. Başlangıç Rehberi

Adım 1: Backend'i Kurma

  1. .NET 5 SDK'nın kurulu olup olmadığını kontrol et: 
    dotnet --version
  2. Backend klasörüne git: 
    cd backend
  3. Veritabanını oluştur: 
    dotnet ef database update
  4. API'yi çalıştır: 
    dotnet run
✅ Başarılı! API şu adreste çalışıyor: 
http://localhost:5000

Adım 2: Frontend'i Açma

  1. frontend.html dosyasını tarayıcıda aç
  2. veya Live Server (VS Code) ile aç
  3. veya Python server ile aç: 
    python -m http.server 8000

Adım 3: Test Etme

  1. Tarayıcı konsolunu aç: F12
  2. "Kayıt Ol" butonuna tıkla
  3. Form doldurup kayıt oluş
  4. Giriş yap ve ilanları test et

5. API Endpoints

Method URL Açıklama

Araçlar (Cars)

Method URL Açıklama Auth
GET /api/cars Tüm ilanlar Hayır
GET /api/cars/{id} Tek ilan Hayır
POST /api/cars Yeni ilan oluştur Evet
PUT /api/cars/{id} İlan güncelle Evet
DELETE /api/cars/{id} İlan sil Evet
GET /api/cars/user/{userId} Kullanıcının ilanları Hayır

7. Kimlik Doğrulama (JWT)

Token Yapısı

Header: {
  "alg": "HS256",
  "typ": "JWT"
}

Payload: {
  "sub": "1",
  "email": "user@example.com",
  "name": "username",
  "iat": 1645353600,
  "exp": 1645440000
}

Signature: HMACSHA256(base64UrlEncode(header) + "." + base64UrlEncode(payload), secret)

Token Kullanımı

GET /api/cars
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

9. Sorun Giderme

❌ Sorun: "dotnet: The term 'dotnet' is not recognized"
✅ Çözüm: .NET 5 SDK kurulu değil. Microsoft .NET website'sinden indir ve kur.
❌ Sorun: "Cannot connect to database"
✅ Çözüm: 
cd backend
dotnet ef database drop
dotnet ef database update
❌ Sorun: "Frontend API'ye bağlanılamıyor"
✅ Çözüm:
  1. Backend'in çalışıp çalışmadığını kontrol et: http://localhost:5000
  2. Firewall ayarlarını kontrol et
  3. Browser konsolunu aç (F12) ve hataları gör
  4. test-api.html ile API'yi test et

11. Güvenlik Notları

  • ✅ Şifreler BCrypt ile hashlenir (11 rounds)
  • ✅ JWT token'lar 24 saat geçerlidir
  • ✅ CORS tüm originlere açıktır (production'da kısıtla)
  • ✅ SQL Injection koruması (Entity Framework kullanımı)
  • ✅ XSS koruması (HTML encoding)

  3. test-api.html ile API'yi test edin
